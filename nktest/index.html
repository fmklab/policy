Политика конфиденциальности


  1. Обзор

    2. «Fmklab» (Разработчик) создал «ВРК-1. Тех.учеба» (Приложение) в качестве приложения с использованием персональных данных. Настоящее Приложение предназначено для проведения централизованного обучения и тестирования специалистов (Пользователь) вагонных ремонтных депо и других структурных подразделений АО «ВРК-1» при проведении технических занятий.

    Эта Политика конфиденциальности используется для информирования о Политике сбора личной информации Пользователя.


  2. Сбор и обмен информацией

    3. 2.1. Использование персональных данных

    Приложение использует фамилию, имя, отчество Пользователя, идентификаторы устройства для осуществления протокола авторизации.


    2.2. Использование камеры

    Приложение использует камеру для верификации Пользователя, который проходит тестирование.


    2.3. Обмен информацией

    Информация, собранная Приложением, используется контролирующей стороной для идентификации Пользователя, мониторинга и анализа результатов обучения и экзаменов.


    Персональные данные:

    * Фамилия, имя, отчество;

    * Идентификаторы устройства;

    * Фотография Пользователя;
